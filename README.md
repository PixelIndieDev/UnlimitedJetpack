# Unlimited Jetpack
A simple mod for Lethal Company that disables the battery drain of the jetpack. 

With this mod installed, the jetpack no longer loses charge, allowing you to use the jetpack indefinitely without recharging.

## Installation
1. Install `BepInEx`
2. Drop 'BepInEx' **from the UnlimitedJetpack.zip** into your Lethal Company folder.

## Usage
No configuration needed.

> [!IMPORTANT]
> All players in the lobby are required to have the `Unlimited Jetpack` mod installed. Clients without the mod will receive an generic "an error occurred" message and be sent back to the main menu.

## Compatibility
This mod only modifies the jetpack battery draining behavior and should work alongside most other mods.
